Three JS Desktop Vision Oauth code Example

This example showcases Desktop Vision running in a threejs scene.

Flow:

  1. Generate oauth code

  2. Enter oauth code into https://desktop.vision/app/#/authorize

  3. The code will automatically be used to retrieve auth token after it is validated, and list available computers on the account

  4. Select a computer to connect to and display in the scene
